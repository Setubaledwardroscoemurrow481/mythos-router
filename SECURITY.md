# Security Policy

## Philosophy
Mythos Router is designed with a "zero-trust AI" model:
AI outputs are never trusted without verification.

## Safe Execution
- Shell commands are restricted via whitelist
- No arbitrary command execution allowed
- Optional paranoid mode disables all shell access

## Environment Variables
- Explicit validation required
- No fallback defaults for sensitive values

## Reporting Vulnerabilities
If you find a security issue, contact:
[https://x.com/thewaltero]

## Scope
This tool is designed for local execution.
Users are responsible for reviewing all AI-generated actions.
